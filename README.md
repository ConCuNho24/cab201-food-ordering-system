# CAB201 Food Ordering System

Object-oriented food ordering system implemented in C#.

## Features
- Multiple user roles (Customer, Deliverer, Client)
- Order creation and management
- Menu system
- Text-based interface
- Separation of UI and business logic

## Tech Stack
- C#
- Object-Oriented Programming
- Console Application
- .NET

## Concepts Used
- Inheritance
- Encapsulation
- Polymorphism
- Abstraction
- Class-based architecture
